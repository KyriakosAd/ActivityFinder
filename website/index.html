This website will help you find things to do when you are bored!

There are fields like activity type, the cost of the activity and more that help you narrow down your results.


You can start by choosing the type of activity from the dropdown menu called Categories.

About

  • The API Call is happening using JavaScript and specifically using a technique called AJAX.

AJAX is a new technique for creating better, faster, and more interactive web applications with the help of XML, HTML, CSS, and JavaScript.
